## 🐛 Reporting Bugs

When reporting a bug, please include:

### Required Information

- **Description** — What happened? What did you expect to happen?
- **Steps to reproduce** — How can I reproduce the issue?
  1. Go to '...'
  2. Click on '...'
  3. See error
- **Environment** — What device/browser are you using?
  - Device: (e.g., iPhone 13, Samsung Galaxy S21, Desktop)
  - Browser: (e.g., Chrome 120, Safari 17, Firefox 121)
  - OS: (e.g., iOS 17, Android 14, Windows 11, macOS 14)

### Optional but Helpful

- **Screenshots** — A picture is worth a thousand words
- **Console errors** — Open browser console (F12) and copy any error messages
- **Expected behavior** — What should have happened instead?
- **Workaround** — Did you find a temporary solution?

### Example Bug Report

```markdown
**Description:**
When I try to add a new weight entry, the save button doesn't work.

**Steps to reproduce:**
1. Go to "Body Weight" tab
2. Enter weight value (e.g., 75.5)
3. Click "Save"
4. Nothing happens

**Environment:**
- Device: iPhone 13
- Browser: Safari 17.2
- OS: iOS 17.2

**Screenshot:**
[Attach screenshot]

**Console errors:**
TypeError: Cannot read property 'userId' of null
```

## 💡 Suggesting Features

Have an idea for a new feature? Great! Here's how to suggest it:

### What to Include

- **Feature description** — What do you want to add?
- **Use case** — Why is this feature useful? What problem does it solve?
- **Examples** — How would it work? Can you provide examples?
- **Priority** — How important is this feature to you? (Low/Medium/High)

### Example Feature Request

```markdown
**Feature:** Export data to PDF

**Use case:**
I want to share my health progress with my doctor, but CSV files are not easy to read. A PDF export with charts would be perfect.

**How it would work:**
- Add an "Export to PDF" button next to "Export CSV"
- Generate a PDF with:
  - Summary of the selected period
  - Charts for each metric
  - Table with detailed data

**Priority:** Medium
```
